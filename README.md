# Data Structures and Algorithms (DSA)

A comprehensive repository for practicing data structures, algorithms, and solving coding problems.

## 📁 Project Structure

```
dsa/
├── algorithms/          # Core algorithm implementations
│   ├── graph/          # Graph algorithms
│   └── trie/           # Trie data structure implementations
├── interviews/         # Interview preparation problems
│   └── quince/         # Company-specific interview problems
├── leetcode-daily/     # Daily LeetCode problem solutions
│   └── src/
│       └── jan_2026/   # Problems organized by month
└── out/                # Compiled Java classes (generated)
```

## 🚀 Getting Started

### Prerequisites

- Java JDK 8 or higher
- IntelliJ IDEA (recommended) or any Java IDE

### Running the Code

1. Clone this repository
2. Open the project in IntelliJ IDEA
3. Navigate to any Java file and run the `main` method

## 📚 Contents

### Algorithms

- **Graph Algorithms**: Graph traversal and path-finding algorithms
- **Trie**: Trie data structure implementations including:
  - Basic Trie operations (insert, search, startsWith)
  - Maximum XOR problems

### Interview Preparation

- Company-specific interview problems and solutions
- Practice problems from various coding interview platforms

### LeetCode Daily

- Daily LeetCode problem solutions organized by month
- Problems solved in January 2026:
  - Minimum Absolute Difference
  - Minimum Difference Between High and Low
  - Minimize Maximum Pair Sum
